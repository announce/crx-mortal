## Getting Started

1. Install the extension at [**Chrome Web Store**](https://chrome.google.com/webstore/detail/crx-mortal/plbhlfecmbmkphfgcpoijlidjapddidj?utm_source=github)
2. Submit your game at https://mjai.ekyu.moe/

## Development

Prerequisites:

* Docker
* Node v18

Run:

```bash
npm run watch
```

For the release:

```bash
npm run build
```
