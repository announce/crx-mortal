## Getting Started

1. Install the Chrome Extension at [**Chrome Web Store**](https://chrome.google.com/webstore/detail/crx-mortal/plbhlfecmbmkphfgcpoijlidjapddidj?utm_source=github)

## Development

Prerequisites:

* Docker
* Node v18

Run:

```bash
./app start
```

For the release:

```bash
./app release
```
